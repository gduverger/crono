Getting Started

1. Install the Crono Python client library:

pip install crono-python

2. Schedule recurring or single occurence jobs in your application (in app.py for example):

import crono

scheduler = crono \
  .Scheduler(token='{token}')

# Timer (single occurrence)
scheduler \
  .in(seconds=60) \
  .request('{url}')

# Datetime (single occurrence)
scheduler \
  .on(<datetime>) \
  .text('hello', 
    from='{phone}', 
    to='{phone}')

# Interval (recurring job)
scheduler \
  .every(minutes=5) \
  .email('hello', 
    from='{email}', 
    to='{email}')

# Crontab (recurring job)
scheduler \
  .every('0 6 * * 2') \
  .run('hello', args=[], kwargs={})

# Decorator
@scheduler.every(hours=1)
def hello(args, **kwargs):
  print('hello')

3. Run the Crono server:

crono app:scheduler

Authentication

Requests to all endpoints have to be authenticated with a Authorization: Bearer {token} header. See the example below for more details. Email {{ crono_support_email }} to request a token.

Pricing

Crono costs $0.01/execution and your first 1,000 executions are free (no payment information required). Email {{ crono_support_email }} to request a token.
