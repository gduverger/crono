Getting Started

1. Install the Crono Python client:

pip install crono-python

2. Schedule recurring or single occurence jobs in your application (in app.py for example):

import crono

scheduler = crono \
  .Scheduler(token='{token}')

# Timer
scheduler \
  .request('{url}') \
  .in(minutes=1)

# Datetime
scheduler \
  .text('hello', 
    from='{phone}', 
    to='{phone}') \
  .on(<datetime>)

# Interval
scheduler \
  .email('hello', 
    from='{email}', 
    to='{email}') \
  .every(hours=1)

# Crontab
scheduler \
  .run('hello', args=[], kwargs={}) \
  .every('0 6 * * 2')

# Decorator
@scheduler.every(days=1)
def hello(args, **kwargs):
  print('hello')

3. Run the Crono server:

crono app:scheduler

Triggers

There are 4 types of triggers: Timer, Datetime, Interval, and Crontab.

Timer represents a countdown until the execution of the task. It takes at least 1 keyword argument: hours, and/or minutes. Those parameters are cumulative.

Client: 
scheduler \
  .in(hours=1, minutes=2) \
  .…
API: 
{
  "trigger": {
    "type": "timer",
    "kwargs": {
      "hours": 1,
      "minutes": 2
    }
  }
}

Datetime specifies the execution of a task at a specific date and time. It only occur once. It is instantiated in the Crono Python client library by calling the method on on the Scheduler object. It takes 1 argument: datetime.

import datetime
independence_day = datetime \
  .datetime(2019, 7, 4)
scheduler \
  .on(datetime=independence_day) \
  .…

Interval specifies a frequency at which to execute a task. It occurs multiple times. It is instantiated in the Crono Python client library by calling the method every on the Scheduler object. It takes at least 1 keyword argument: hours, and/or minutes.

scheduler.every(hours=1).…

Crontab takes a crontab expression to specify the execution time. It is instantiated in the Crono Python client library by calling the method every on the Scheduler object.

scheduler.every('0 6 * * 2').…

  .Scheduler(token='{token}')
API:
