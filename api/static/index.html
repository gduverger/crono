Introduction

Crono lets you trigger (e.g., interval, date, cron) commands (e.g., get, post, email, text) and monitor jobs.

Authentication

Every request needs to be authenticated by providing a token in the header.

Endpoints

GET /jobs

bash $ curl -X GET https://www.crono.com/{version}/jobs \ -H "Authorization: Token {token}" HTTP/1.1 200 OK [{ "job": { "id": "060311fabafa42c3a8aee47905f5a2d1", "name": "email", "command": "<function email at 0x7f2706ec02f0>", "trigger": "<IntervalTrigger (interval=datetime.timedelta(0, 3600), start_date='2018-01-19 02:38:34 UTC', timezone='Etc/UTC')>", "args": ["Bonjour !"] } }]
